# MIMIC-IV 30-Day Hospital Readmission Prediction

Capstone project predicting 30-day hospital readmission risk using the MIMIC-IV clinical database (PhysioNet credentialed access required). Includes EDA, SQL-based cohort extraction, feature engineering, and machine learning classification pipeline.

## Team
- Dharani Kommideddi
- Alekhya Konakanchi (AlekhyaaKonakanchi)

## Tools
Python, SQL, Pandas, Scikit-learn, Healthcare Analytics

## Files
- main.py: main pipeline script
- eda.py: exploratory data analysis
- exploration.sql: SQL cohort extraction queries

## Data Access
This project uses restricted MIMIC-IV data via PhysioNet credentialed access.
No patient data or raw MIMIC-IV files are included in this repository.
Collaborators must obtain independent PhysioNet credentials before use.
